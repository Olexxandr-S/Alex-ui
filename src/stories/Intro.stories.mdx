# Alex UI

Welcome to "Alex UI" - the ultimate React UI library!

## Installation

To get started with "Alex UI", simply install it via npm or yarn:

```
npm install alex-ui
```

or

```
yarn add alex-ui
```

## Usage

To use "Alex UI" in your React project, simply import the components you need and render them in your code. 
Here's an example:

```
import { Button } from 'alex-ui';

const App = () => {
  return (
    <Button>Click me!</Button>
  );
};
```
